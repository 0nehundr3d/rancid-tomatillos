# Rancid Tomatillos - Starter Repo

## Set Up

Frontend:
- Clone Repo
- `cd` into directory
- `npm i`
- `npm start`
- Navigate to `localhost:3000` in your browser to see the frontend

Backend:
- Follow the set up instructions for [this repo](https://github.com/turingschool-examples/rancid-tomatillos-api/)
